# 这个工具到底是干啥的（大白话版）

## 一句话

**给工程师做"这个新材料能不能换上去"的快速初筛。10 分钟告诉你"能用/边缘/不能用"，并解释为什么——避免直接做 1 个月的台架实验。**

## 真实使用场景

研发场景：**你想知道一个新材料 X 能不能替换现有零件 Y 上的材料 Z**。

**没这个工具之前的工作流（痛点）：**

1. 查 5-10 份供应商资料、教材、行业标准 → 半天
2. 用 Excel 算应力、应变、安全系数 → 半天
3. 凭工程经验拍脑袋判断"应该可以" → 不严谨
4. 写报告说"建议做实验" → 1-2 天
5. **实际做实验失败，发现是常识级问题**（比如材料根本不耐高温）→ 浪费 1 个月

**有这个工具之后：**

1. UI 选材料 + 选零件 + 输入工况（载荷、温度、几何）→ 30 秒
2. 系统 10 秒出报告：
   - 这个材料在你这个工况下**能不能用**（红/黄/绿）
   - 安全系数是多少（带不确定度区间，不撒谎）
   - 哪些证据支持这个判断
   - 工况是否在材料适用范围（超出会拒绝出数字）
3. 拿这份报告 → 决定要不要做实验

## 它能解决什么、不能解决什么

### ✅ 能解决（节省工程师时间）

| 痛点 | 工具怎么帮 |
|---|---|
| **不知道这个材料数据靠不靠谱** | 每条数据带 confidence 和来源，区间式数值（low/typical/high）反映不确定度 |
| **担心算错应力公式** | 内置 5 种零件拓扑的公式（梁/工字梁/板/带/波纹），都经过测试 |
| **复合材料失效不会判** | 自动跑 Tsai-Wu（碳纤维板）或 von Mises（金属/塑料） |
| **怕用了工况超出材料范围** | 工况包络校验：温度/载荷/厚度等超范围直接拒绝出数 |
| **不知道有没有更合适的替代材料** | 工况超范围时，自动反查库内**哪些材料覆盖你的工况** |
| **担心数据来源说不清** | 报告每条结论都绑定计算公式或证据卡 |

### ❌ 不能解决（明确边界）

| 限制 | 为什么 |
|---|---|
| **不能替代 FEA 仿真** | 用的是闭式公式，复杂应力场算不准。这是"初筛"不是"认证" |
| **不能给出量产/认证结论** | 工程默认参数（如 Tsai-Wu F12 耦合系数）还没用真实实验校准 |
| **不能算疲劳寿命** | 没有 S-N 数据库（Phase 3 候选） |
| **不能算屈曲/热应力** | Phase 3 候选 |
| **数据库材料只有 12 个** | MVP 验证用；正式版要扩充到 50+ |

## 工程师 5 分钟上手指南

### Step 1：选评估对象（左侧 Sidebar）

- **应用领域**：8 大行业（航空、机器人、医疗、汽车……）
- **核心零部件**：每个行业 1-3 个零件模板（自动决定计算拓扑）

### Step 2：选材料

3 种模式：
- **单一均质材料**：自己输密度/强度/模量（金属、塑料）
- **从基准材料库选择**：库里 12 个常用材料（推荐——能用三点区间和工况适用域）
- **复合/杂化材料体系**：基体+增强体配比，跑混合定律

### Step 3：输入工况

- 几何尺寸（mm）
- 反向载荷（可选）：轴向力 + 弯矩 — **输了就能跑"安全系数"分析**
- 温度（在材料详情里隐式）

### Step 4：跑"运行内部初筛"

10 秒后看 6 个 tab：

| Tab | 看什么 |
|---|---|
| **工程初筛** | 重量/抗弯/抗剪/挠度等核心指标，带不确定度图 |
| **证据卡** | 系统从内部资料库召回的支撑证据 |
| **中文报告** | 完整 markdown 报告，可下载/打印 |
| **评估记录** | 历史评估列表，可回看 |
| **检索评估** | 测试证据库召回质量 |
| **安全性评估** | 安全系数区间 + pass/marginal/fail 结论 |

### Step 5：看结论

- **🟢 pass**（SF.low ≥ 1.5）：放心做下一步实验
- **🟡 marginal**（1.0 ≤ SF.low < 1.5）：值得做实验验证
- **🔴 fail**（SF.typical < 1.0）：换材料或改设计
- **⛔ refusal**：工况超出材料适用域，看替代材料建议

## 当前可用的 3 个真实场景

工具内置 3 个 MVP 验证场景（其余场景也能跑，但 envelope/强度数据未校准）：

1. **人形机器人核心骨架 / 下肢大扭矩管状连杆** + Ti-6Al-4V
2. **智能穿戴 / 智能穿戴承力外壳** + PA66-GF30
3. **智能穿戴 / 柔性外骨骼助力带** + Kevlar

## 跟 legacy 原型相比的取舍

Legacy 原型（拆解到 `legacy/app_legacy_streamlit.py`）有几个亮眼的功能在当前 MVP 中**暂时**没有：

| Legacy 有 | 当前 MVP 没有 | 是否值得恢复 |
|---|---|---|
| 3D 结构蓝图（双引擎宏观+微观） | ❌ | **值得**——视觉冲击对汇报有用 |
| Tavily 联网搜索 | ❌ | 一般——内部资料库已能用 |
| 商业级八维剖析 | ❌ | 一般——重业务展示，工程价值低 |
| 最终红/黄/绿决议总结 | ❌（分散到 6 个 tab） | **值得**——一眼看结论 |

**Phase 3 候选**（按 ROI 排序）：

1. **顶部"评估总览"面板**：把当前散落在 6 个 tab 的关键信息（pass/fail、SF 区间、控制层、关键风险）汇总成一页，工程师/老板**一眼看懂**
2. **3D 蓝图回归**：每个零件拓扑配一个简化 3D 几何展示
3. Hashin 分模式失效（复合材料更精细的失效判定）
4. 知识库扩充
